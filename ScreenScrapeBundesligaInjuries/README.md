# PIN Guesser
Shows how insecure a 4 digit pin is.


## Setup
### On Mac:
1. Run
```
brew install geckodriver
```
2. Run
```
pip install -r requirements.txt
```


## Run it
To run the program call the script.

```
./pin.py
```

These are the foundations.
